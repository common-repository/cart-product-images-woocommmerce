Manage Cart Images through on of button in admin panel 

== Description ==

By default, the WooCommerce cart page includes product images and links to product pages in your shop for each item in the cart.  by using this plugin you can enable and disable these images. 
for more help in documentation <a href='https://xpertcodes.com' >https://xpertcodes.com</a>

or you can email me as well 12345adnan@gmail.com
See screenshots


== Installation ==
    Install the plugin
    Activate the plugin
    Navigate to WC CPIW in admin menus
    Enable yes or no
